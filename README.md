# NanoSepex Website

A static NanoSepex marketing site built from the supplied PowerPoint design.

## Local preview

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

## Vercel

This project is dependency-free static HTML, CSS, and JavaScript. Import the folder into Vercel and deploy with the default static settings.
